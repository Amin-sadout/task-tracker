# Task Tracker

# Overview
This is a beginner JavaScript project I built to practice DOM manipulation, arrays, objects, and rendering dynamic content on the page.

The app lets the user:
add tasks
mark tasks as complete
undo completed tasks
delete tasks
track total, completed, and remaining tasks


# What I Practiced

### JavaScript

boolean toggling
more use of array methods like push() and splice()

# Problem Solving
validating empty input
trimming user input with .trim()
storing task data as objects
counting completed tasks
calculating remaining tasks
toggling task completion on and off
